02. Installation

Theme Requirements

In order to proceed with the theme installation, you need to make sure you have the latest version of WordPress (4.0.1) on your self-hosted site.

Theme Installation

  1. Log into your WP Admin and browse for Appearance → Themes.
  2. Now go to the secondary tab: Install Themes and click the Upload link option.
  3. Follow the instructions by choosing the file "jovial.zip", located in the archive which you have downloaded and click "Install Now" button.
  4. Click Activate link, once the upload has finished, and you are ready for the next steps.

Media Settings - Important

Please change your media settings to the one as described below.

Althugh these settings are not mandatory buy highly recommended in order for the post images to work properly.

It is important to upload images in these propotions only as this will save your bandwidth and disk space.

  1. Thumbnail Size - 150 by 66
  2. Medium Size - 275 by 175
  3. Max width - 800 by 400

Recommended Plugins

We recommend you to install these plugins although they are not compulsory.

W3 Total Cache - Easy Web Performance Optimization (WPO) using caching: browser, page, object, database, minify and content delivery network support.

WordPress SEO by Yoast - Improve your WordPress SEO: Write better content and have a fully optimized WordPress site using the WordPress SEO plugin by Yoast..

03. Homepage Setup

Prepare the page

  1. Go to: Pages → Add New.
  2. Enter the title for this page: Homepage or Home, whatever you like.

Front Page Displays

Now you have to tell WordPress wich page is your site homepage.

  1. Go to: Settings → Reading.
  2. In a "Front Page Displays" select "A static page (select below)"
  3. In a "Front page:" dropdown select your "Homepage" page.
  4. You are done! Now you have your Homepage.

04. Adding New Post & Pages

To create a new post go to Posts → Add New.

Similarly, To start creating the pages go to Pages → Add New.

Static page can be created with sidebar or full width (without the sidebar).

Locate "Page Attributes" box on the right side of the page.

1. Default - Leave the "Default Template" in "Template" dropdown to create page with sidebar.
2. Full Width - Select the "Full Width" option to use the page as full width page.
3. Author's List - Select the "Author's List Template" option to show complete list of guest contributers on the page. This page is full width by default.
By default, the list will only display authors with role - Contributers. Please change your guest author's role to contributers incase you have assigned them some other role.

05. Navigation Setup

Setting Up New Menus

Follow the next steps to define menus for each of the theme's menu locations. In each menu, you can add as many items you need.

  • Go to Appearance → Menus and click - Create a New Menu.
  • Give it a name, add links and click "Save Menu" button.
  • Assign a Menu Location for your newly created menu, by selecting it for the preferred Theme Location.

Menu Locations

The theme comes with 2 custom menu locations:

  • Main Menu
  • Footer Menu

Main Menu "Mega Menu" Feature

By default the theme supports Mega Menu feature in the main menu. You don't have to do anything extra to display anything in mega menu.

Only make sure that the child items of mega menu are in even ratio.

Incase you don't want to display Mega Menu.

Incase you don't want to display Mega Menu and wanted to use the ordinary style only;

Just add a CSS class "simplemenu" without quotes for the parent list item

06. Theme Options

This theme uses inbuilt WordPress Customization functions for all customization purposes.
The best thing about this kind of panel is that you can view your changes on the right side as soon as you make them without hitting the save button.

You can easily take control over your site, the panel allows you to:

  • Title & Tags
  • Logo & Favicon
  • Colors
  • Navigation
  • Copyright Notes
  • Social Links
  • Sidebar Position

To access the Theme Options panel, go to Appearance → Customize.

Changing The Site Title

Go to the "Site Title & Tagline" tab and click the 'Save & Pubish' button.

Changing Logo & Favicon

Go to the "Logo & Favicon" tab, Upload your logo and favicon and click the 'Save & Pubish' button.

Changing Site Colors

Go to the "Colors" tab. Panel offers you different color selectors for differnt sections. Use accordingly. Click the 'Save & Pubish' button.

Changing Social Links

Go to the "Social Links" tab. Enter your social profile links. Click the 'Save & Pubish' button.

Changing Sidebar Position

Go to the "Sidebar Position" tab. You can choose to have your sidebar on either Left or Right. Click the 'Save & Pubish' button.
